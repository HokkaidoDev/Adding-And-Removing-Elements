# Adding-And-Removing-Elements

## Description

This is an educational project demonstrating how to add and remove elements from the DOM using JavaScript. It's a simple TODO application where users can add new tasks to a list and remove them by double-clicking on the item or using a delete button.

The project showcases two approaches to adding elements:

1. Using `document.createElement()` and `append()` methods
2. Using `insertAdjacentHTML()` for inserting HTML strings

## 🌐 Demo

[View Live Demo](https://hokkaidodev.github.io/Adding-And-Removing-Elements/)

## Features

- Add new TODO items via a form
- Remove items by double-clicking on them
- Remove items using the 'x' button next to each item
- Form resets automatically after adding an item

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)

## How to Run

1. Open `index.html` in your web browser
2. Enter a TODO item in the input field
3. Click the "Add" button to add it to the list
4. Double-click on an item or click the 'x' button to remove it

## Educational Notes

This project illustrates fundamental DOM manipulation techniques in JavaScript. The code includes comments explaining the two different approaches to adding elements, with recommendations for modern development practices.
